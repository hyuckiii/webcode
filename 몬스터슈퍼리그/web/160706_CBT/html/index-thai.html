รางวัลการลงทะเบียนเข้าร่วม Exclusive Test และร่วมตอบแบบสอบถาม
จะถูกส่งให้ทางข้อความโทรศัพท์ในรูปแบบคูปอง ส่วนรางวัลล็อคอินต่อเนื่อง 3 วันและ 8 วัน
จะได้รับทางกล่องจดหมายภายหลังเกมเปิดให้บริการ

กรุณาเชื่อมต่อเฟสบุ๊คหรือกูเกิ้ลในช่วง Exclusive Test
เพื่อรับของรางวัลเมื่อเชื่อมต่อด้วยบัญชีเดิมตอนเกมเปิดให้บริการ
  • ประกาศเกี่ยวกับรายละเอียดการจัดกิจกรรม Exclusive Test จะถูกส่งผ่านทาง SMS
  • รางวัลการลงทะเบียนเข้าร่วม Exclusive Test และรางวัลล็อคอิน
    จะถูกจัดส่งให้ทางข้อความโทรศัพท์และกล่องจดหมายในเกม
    ภายหลังเกมเปิดให้บริการ
  • หากพบมีการแอบอ้างหรือขโมยข้อมูลของผู้อื่น จะไม่ได้รับรางวัล และอาจมีความผิดทางกฎหมาย
  • ข้อมูลเกมทั้งหมดจะถูกรีเซ็ตเมื่อจบ Exclusive Test
  • ผู้ใช้แอนดรอยเท่านั้นที่สามารถเข้าร่วมได้
  • ข้อมูลที่เก็บได้จะไม่ถูกใช้นอกเหนือจากวัตถุประสงค์ที่กล่าวมา
  • เมื่อกรอกหมายเลขโทรศัพท์เพื่อลงทะเบียนแล้ว จะถือเป็นการเข้าร่วมกิจกรรมทั้งที่เกี่ยวกับ Exclusive Test
  • หากมีการเข้าร่วมกิจกรรมด้วยวิธีที่ไม่ปกติ อาจทำให้ถูกยกเว้นจากการเข้าร่วมกิจกรรมได้
